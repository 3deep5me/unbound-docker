# Security Policy

## Supported Versions

All supported "latest" versions and their corresponding version will be updated as soon as possible in case a vulnerability was found.

| Version   | Supported          |
| --------- | ------------------ |
| latest    | :white_check_mark: |
| 1.15.0rc1 | :x:                |
| 1.14.0    | :white_check_mark: |
